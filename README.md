# Discord All-in-One Professional Bot

A professional Discord bot with:

- Moderation
- AutoMod
- Tickets
- Invite Tracker
- Levels / XP
- Giveaways
- Verification
- AntiNuke
- Welcome / Goodbye
- MongoDB Database

## Setup

1. Install Node.js
2. Run:

npm install

3. Fill `.env`

4. Start bot:

npm start

## Hosting

Use Railway / Oracle Cloud / VPS for 24/7 hosting.

## Commands

Use `/help` to see all commands.
